# 🛒 E-commerce Order Database (SQL)

This project simulates an e-commerce platform database with customers, products, orders, payments, and shipping. It provides various business insights using SQL queries.

## 📦 Database Tables
- `customers`
- `products`
- `orders`
- `order_items`
- `payments`
- `shipping`

## 📊 Insights Performed
- Total revenue
- Monthly sales
- Best selling products
- Top customers
- Order status analysis

## 🧠 Tools Used
- MySQL / PostgreSQL / SQLite
- GitHub

## 📁 Files
- `ecommerce_order_db.sql` – Table creation and sample data
- `queries.sql` – SQL queries for insights
- `README.md` – This documentation
